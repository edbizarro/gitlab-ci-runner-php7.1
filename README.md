## Based on Ubuntu 16.04

### This image contains:

- PHP 7.1 with: ```mcrypt```, ```mongo```, ```xdebug```, ```zip```, ```xml```, ```mbstring```, ```curl```, ```json```, ```mysql```, ```gd```, ```imap``` and ```tokenizer``` extensions
- [Composer](https://getcomposer.org/)
- [Yarn](https://yarnpkg.com)
- Node 7.x / NPM / Gulp
